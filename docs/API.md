# LumBarong API Documentation

This directory contains documentation for the LumBarong API and system architecture.

## API Overview
LumBarong uses a RESTful API built with Laravel (PHP).

### Base URL
- Local: `http://localhost:8000/api/v1`
- Production: (Configure via environment variables)

### Core Modules
- **Auth**: User registration, login, and role-based access control.
- **Products**: Management of authentic Filipino attire and accessories.
- **Orders**: Shopping cart and order processing workflow.
- **Messaging**: Real-time artisan-to-customer chat.
